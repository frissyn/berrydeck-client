# BerryDeck

This repository is the source for the rewrite of BerryDeck.

The new developement stack:

+ **Frontend:** Vue.js v3 (DaisyUI + Tailwind.css)
+ **Server:** Nuxt.js v3
+ **GUI Bridge:** Neutralino.js
+ **Backend:** Node.js v17

Compared to the old developement stack:

+ **GUI Framework:** Slab
+ **GUI Bridge:** Love2D
+ **Backend:** Lua 5.4
